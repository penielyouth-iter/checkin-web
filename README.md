### How to run

    * Clone repo
    git clone https://github.com/penielyouth-iter/checkin-web.git
    cd checkin-web

    * Install dependencies:
    npm install

    * Run locally for debug
    npm run dev

    * Build for production
    npm run build
    
    * Deploy to github pages
    npm run deploy
